This project perfomr an end to end data analysis and machine learning pipeline on the QS World University Ranking 2025 dataset
****It includes:
+ Data cleaning and preprocessing
+ Exploratory Data Analysis (EDA)
+ Unsupervised learning (clustering universities)
+ Supervised learning (predicting Top 50 universities)
## ⚙️ Tech Stack
- Python  
- pandas, numpy  
- matplotlib  
- seaborn  
- scikit-learn  
- pathlib  

### Machine Learning Models
- K-Means Clustering (Unsupervised Learning)  
- Logistic Regression (Classification)  
- Random Forest Classifier (Ensemble Learning)  

### Techniques Used
- Data Cleaning & Preprocessing  
- Missing Value Imputation (Median)  
- Feature Engineering  
- Data Visualization (Boxplots, Heatmaps)  
- Feature Scaling (StandardScaler)  
- Hyperparameter Tuning (GridSearchCV, RandomizedSearchCV)  
- Cross Validation (StratifiedKFold)

## 📊 Key Insights
- Universities cluster into distinct performance tiers based on academic and research indicators
- Reputation and research strength are strong predictors of Top 50 ranking
- Random Forest slightly improves performance over Logistic Regression
- Feature engineering significantly improves model interpretability

---

## 🧠 Model Performance
- Logistic Regression: Optimized using GridSearchCV  
- Random Forest: Optimized using RandomizedSearchCV  
- Evaluation metric: **F1-score (handles class imbalance)**
